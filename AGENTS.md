# AGENTS.md

이 저장소에서 에이전트는 아래 규칙을 따른다.

## 우선순위
1. 사용자 명령
2. 프로젝트 문서 규칙 (`docs/**` 포함)
3. 에이전트 판단

## 절대 금지
- 사용자의 명시적 요청 없이 빌드 실행 금지

## 작업 규칙
- 쓰기 작업 전에 변경 대상 파일을 먼저 명시한다.
- 검증/점검은 가능한 경우 읽기 전용 방식(`--dry-run` 등)을 우선 사용한다.
- 금지사항과 충돌 가능성이 있으면 즉시 작업을 멈추고 사용자 확인을 먼저 받는다.

## 현재 기준
- 에셋은 현재 `asset/*.json` combined 구조를 공식 기준으로 사용한다.
- `src.json`, `manifest.json`, `study.json` 기반 마이그레이션 계획은 폐기된 기준으로 본다.
- 현재 에셋 구조는 `docs/CURRENT_ASSET_STRUCTURE.md`를 따른다.
